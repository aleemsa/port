Projects

Color Cascade (Unity 2D)

In Color Cascade, I designed a vibrant and engaging puzzle game that features a progressive difficulty curve and matching mechanics. My responsibilities included developing the core gameplay logic, implementing the level manager, and creating a responsive scoring system using C#. To ensure a smooth player experience, I conducted thorough user testing, which helped refine the UI and balance the game’s overall challenge.

First-Person Shooter (Unity 3D)

This project involved building a dynamic first-person shooter from the ground up. I created a custom player controller and implemented intelligent enemy AI and responsive gun mechanics. The game also includes immersive sound effects, dynamic lighting, a health system, and respawn functionality. Unity’s built-in Physics engine was leveraged to deliver realistic bullet behavior and accurate collision detection, enhancing the overall gameplay experience.

Space Shooter Game (Unity 2D/3D)

I developed a hybrid 2D/3D vertical scrolling space shooter game that challenges players with waves of enemies and boss fights. The game features optimized enemy spawning logic using prefabs and object pooling, which significantly improved performance. To enhance the visual experience, I implemented parallax scrolling backgrounds and various power-ups, adding strategic depth and replayability to the game.
